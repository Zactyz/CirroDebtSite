# Cirro Debt - Community Debt Payoff Platform

A modern, mobile-friendly website for Cirro Debt, a platform that helps users collectively tackle debt through pooled payments.

## Project Overview

This website serves as the main landing page for Cirro Debt, showcasing the platform's features, benefits, and how it works. The design prioritizes:

- Clean, conversion-focused UI/UX
- Mobile responsiveness
- Fast loading times
- Persuasive copywriting
- Dark/light mode toggle

## Features

- Responsive layout that works on all device sizes
- Dark mode support with system preference detection and manual toggle
- Smooth scrolling navigation
- SVG icons for crisp display at any resolution
- Optimized images with lazy loading

## Setup & Deployment

### Local Development

1. Clone the repository
2. Open `index.html` in your browser to view the site locally

### Deployment to GitHub Pages

To deploy the site to GitHub Pages:

1. Push the repository to GitHub
2. Go to repository Settings > Pages
3. Select the main branch as the source
4. Save the changes to trigger deployment

## Color Scheme

The website uses a carefully designed color scheme with light and dark mode variations:

### Base Colors

- Primary: `#ff9b0e` (orange-peel)
- Primary Alt: `#fd9e0e` (orange-peel-2)
- Accent: `#c17f15` (dark-goldenrod)
- Secondary: `#0a1d2e` (oxford-blue)

The complete color scheme is defined in the CSS variables in `styles.css`.
